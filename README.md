## 💻 Como baixar o projeto 

```bash
#Clonando o repositorio 
$ git clone https://github.com/SamiraFreitas/Projeto_Calculadora_React
#Entrar no diretorio 
$ cd Projeto_Calculadora_React
# Instalar as dependências
$ npm install || yarn install
#Rode o projeto 
yarn start

```
